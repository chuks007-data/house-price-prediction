# House Price Prediction Project

## Overview

This project uses machine learning to predict house sale prices using housing market data.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model Performance

- MAE: 16,775.85
- RMSE: 22,110.57
- R² Score: 0.9629
